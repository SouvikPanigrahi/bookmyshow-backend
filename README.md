## BookMyShow Backend
Backend system for a BookMyShow-like application built using Spring Boot.

# Features
Movie, show, and seat management APIs
Transactional seat booking with concurrency handling
Booking and cancellation workflows
RESTful APIs with clean DTO design
MySQL database integration using Spring Data JPA
#Tech Stack
Java
Spring Boot
Spring Data JPA
MySQL
Maven
# Key Concepts
Database transactions
Concurrency control
Exception handling
REST API design
